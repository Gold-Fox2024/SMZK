# 模糊综合评价 (Fuzzy Comprehensive Evaluation)

模糊综合评价法是一种基于模糊数学的综合评价方法。该综合评价法根据模糊数学的隶属度理论把定性评价转化为定量评价。

## 基本思想
1. 确定因素集 $U = \{u_1, u_2, \dots, u_n\}$。
2. 确定评语集 $V = \{v_1, v_2, \dots, v_m\}$。
3. 建立模糊关系矩阵 $R$。
4. 确定权重向量 $A = (a_1, a_2, \dots, a_n)$。
5. 合成模糊综合评价结果 $B = A \circ R$。
